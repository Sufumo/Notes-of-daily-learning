# 欢迎来到Astro博客

这是一个使用 **Astro** 构建的静态博客示例。

## 什么是Astro？

Astro 是一个现代的静态站点生成器，它专注于：

- 🚀 **性能优先** - 默认情况下不发送JavaScript
- 📦 **组件化** - 支持React、Vue、Svelte等框架
- 🎨 **内容驱动** - 内置Markdown和MDX支持
- ⚡ **快速构建** - 构建速度极快

## 开始使用

这个博客使用了Astro的Content Collections功能来管理博客文章。所有文章都存储在 `src/content/blog/` 目录中。

你可以通过运行以下命令来启动开发服务器：

```bash
npm run dev
```

然后访问 `http://localhost:4321` 查看你的博客。
